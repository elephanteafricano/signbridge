# SignBridge

SignBridge est une plateforme de traduction bidirectionnelle entre la Langue des Signes Française (LSF) et le français écrit/oral.  
Elle permet :  
- la reconnaissance de signes via caméra ou vidéo,  
- la génération d’un avatar signant à partir d’un texte ou d’un audio,  
- un mode apprentissage avec dictionnaire, leçons et correcteur basé sur la caméra.
